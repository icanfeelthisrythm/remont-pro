## 📋 О проекте

**РемонтПро** — это современный одностраничный сайт для компании, занимающейся ремонтом квартир. Проект демонстрирует владение HTML, CSS и JavaScript, включая работу с API, валидацию форм и создание интерактивных компонентов.

### ✨ Ключевые особенности

- 📱 **Полная адаптивность** — корректное отображение на всех устройствах (mobile-first подход)
- 🎨 **Уникальный дизайн** — тема "чертеж + строительная разметка" с тёплой цветовой палитрой
-  **Видео-карусель** — переключение видео с превью и счётчиком
- 🖼️ **Галерея работ** — слайдер с поддержкой свайпов на мобильных
- 📊 **Калькулятор стоимости** — динамический расчёт с учётом дополнительных опций
-  **Две формы обратной связи** — отправка через Formspree с валидацией
- 🎯 **Интерактивная схема** — маркеры с тултипами на плане квартиры
- 🌓 **Плавная навигация** — якорные ссылки и эффекты при скролле

## 🔧 Функциональность

### 1. Калькулятор стоимости
- Ввод площади квартиры
- Выбор дополнительных опций (дизайн-проект, перепланировка, закупка материалов)
- Мгновенный пересчёт итоговой суммы
- Валидация: нельзя ввести отрицательное значение или 0

### 2. Формы обратной связи
**Форма расчёта стоимости:**
- Поля: площадь, телефон (с маской), ФИО
- Чекбоксы дополнительных услуг
- Валидация всех полей перед отправкой
- Inline-ошибки (красный текст под полем)
- Отправка на Formspree
- Модальное окно подтверждения

**Форма обратного звонка:**
- Открывается по кнопке в шапке
- Поля: имя, телефон
- Отправка на тот же Formspree endpoint
- Уведомление об успехе

### 3. Видео-карусель
- 6 видео с разными комнатами
- Переключение стрелками или цифрами-счётчиком
- Клик по карточке → открытие видео в модалке
- Клик по категориям → переход к видео и автоматическое переключение

### 4. Галерея работ
- 6 слайдов с изображениями
- Навигация стрелками
- Свайпы на мобильных устройствах (touch events)
- Подписи к каждому слайду

### 5. Табы "Комплектация"
- 3 варианта: Стандарт, Комфорт, Премиум
- Плавное переключение контента
- Анимация появления

### 6. Интерактивная схема
- План квартиры с маркерами
- При наведении/клике показываются тултипы
- На мобильных — тап для открытия/закрытия

##  Адаптивность

Сайт корректно отображается на всех разрешениях:

- **Desktop** (>1024px) — полная версия
- **Tablet** (768px–1024px) — 2 колонки в категориях

## 🔐 Безопасность

### Важное замечание
В проекте используется **Formspree** для отправки форм. Это безопасный метод для статических сайтов, так как:
- Токен не хранится в коде
- Formspree обрабатывает данные на своей стороне
- Можно настроить spam-фильтры и ограничения

**Не рекомендуется** хранить токены Telegram Bot API или другие секреты во фронтенд-коде — это учебный проект, и в реальном приложении такие данные должны храниться на сервере.
